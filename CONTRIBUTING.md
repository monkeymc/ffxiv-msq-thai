# ร่วมแปล FFXIV MSQ Thai — คู่มือสำหรับผู้มีส่วนร่วม

ขอบคุณที่สนใจช่วยตรวจทานคำแปลภาษาไทยของ FFXIV Main Scenario Quest!  
คุณ **ไม่จำเป็นต้องมีความรู้ด้านเทคนิค** เลย — ทำได้ผ่านเว็บเบราว์เซอร์ธรรมดา

---

## ขั้นตอนการช่วยแปล (ทำครั้งเดียว)

### 1. สร้างบัญชี GitHub (ถ้ายังไม่มี)

1. เปิด [github.com](https://github.com) แล้วกด **Sign up**
2. กรอก email, username, password แล้วยืนยันบัญชี

### 2. เข้า GitLocalize

1. เปิดลิงก์ GitLocalize ของโปรเจกต์นี้: **[gitlocalize.com/repo/10779](https://gitlocalize.com/repo/10779)**
2. กด **Sign in with GitHub** แล้วอนุญาตให้ GitLocalize เข้าถึง GitHub ของคุณ
3. ระบบจะพาคุณเข้าหน้าแปลโดยอัตโนมัติ

---

## วิธีใช้งาน GitLocalize

### หน้าแรก

หลังจาก login แล้วคุณจะเห็นรายการไฟล์ Quest ทั้งหมด  
แต่ละไฟล์จะแสดงเปอร์เซ็นต์ที่ตรวจทานแล้ว — **เลือกไฟล์ที่มีเปอร์เซ็นต์ต่ำ** เพื่อช่วยก่อน!

### หน้าแปล

| ซ้าย | ขวา |
|------|-----|
| ข้อความต้นฉบับ (ภาษาอังกฤษ) | คำแปลภาษาไทย ที่คุณแก้ได้ |

- **ข้อความที่มี 🤖 AI** = แปลโดย AI อัตโนมัติ รอการตรวจทาน
- **ข้อความที่มี 👥 Community** = ตรวจทานโดยชุมชนแล้ว ✅

### วิธีตรวจทาน

1. อ่านข้อความต้นฉบับภาษาอังกฤษ (คอลัมน์ซ้าย)
2. อ่านคำแปล AI ในคอลัมน์ขวา
3. ถ้าคำแปลถูกต้องและเป็นธรรมชาติ → **กด "Approve"** (ปุ่มติ๊กถูก)
4. ถ้าอยากแก้ไข → **คลิกที่ช่องคำแปล** แล้วพิมพ์แก้ จากนั้นกด **Save**

> **เคล็ดลับ:** ดู [glossary.md](characters/glossary.md) สำหรับคำศัพท์เฉพาะของเกม  
> และ [characters/](characters/) สำหรับสไตล์การพูดของตัวละครแต่ละตัว

### ส่งผลงาน

เมื่อแปล/ตรวจทานเสร็จในไฟล์หนึ่งๆ:

1. กด **"Create Pull Request"** ใน GitLocalize
2. GitLocalize จะสร้าง Pull Request ให้อัตโนมัติ
3. ระบบ GitHub Actions จะตรวจจับการเปลี่ยนแปลงและอัปเดต tag `AI → Community` ให้เอง
4. ทีม maintainer จะ review และ merge ให้ภายใน 1-2 วัน

## หลักการแปลที่ดี

- **เน้นความเป็นธรรมชาติ** มากกว่าการแปลตรงตัว
- **รักษาอารมณ์และโทนของตัวละคร** เช่น Baderon พูดสำเนียงชาวเรือ, Y'shtola พูดสุภาพ
- **คงชื่อเฉพาะ** ไว้เป็นภาษาอังกฤษ เช่น Eorzea, Grand Company, Ifrit
- **ดูบริบทรอบข้าง** — อย่าแปลประโยคเดี่ยวๆ โดยไม่ดูบรรทัดก่อนและหลัง
