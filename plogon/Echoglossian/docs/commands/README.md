# Plugin Commands

This folder contains the user-facing command reference for Echoglossian.

## Available Commands

- [`/eglo`](./eglo.md) opens the plugin configuration window.
- [`/eglodbmanager`](./eglodbmanager.md) opens the database editor window.
- [`/egloaddonprobe`](./egloaddonprobe.md) starts or stops a recursive addon structure probe.
- [`/egloquestprobe`](../quest-probe-command.md) inspects a quest's Lumina data, live progress, and matching `QuestPlate` rows.

## Notes

- Probe commands are diagnostic tools. They are meant to be used while inspecting logs, addon trees, and database state.
- The configuration and database editor commands are convenience entry points for the plugin UI.
